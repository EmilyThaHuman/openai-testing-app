# OpenAI Testing App

OpenAI Testing App is a comprehensive React-based application that provides a robust platform for testing and implementing various AI capabilities. Built with modern web technologies, it offers a suite of features for working with OpenAI's APIs and other AI services.

## Key Features

- **Chat Interface**: Real-time chat functionality with AI models
- **AI Assistants**: Create and manage AI assistants with specific capabilities
- **Image Generation**: Create and edit images using AI models
- **Audio Processing**: Handle audio transcription and generation
- **Open Canvas**: A flexible workspace for AI experimentation
- **Database Integration**: Supabase integration for data persistence
- **Multi-Provider Support**: Integration with various AI providers including OpenAI, Perplexity, and more

## Tech Stack

- **Frontend**: React with Vite
- **Styling**: Tailwind CSS with shadcn/ui components
- **State Management**: Zustand and Context API
- **API Integration**: Axios with custom middleware
- **Testing**: Vitest with React Testing Library
- **Documentation**: Mintlify

## Project Structure

```

├── actions/       # Action creators and handlers
├── ai/           # AI-related configurations and utilities
├── api/          # API integration layer
├── components/   # Reusable React components
├── context/      # React context providers
├── hooks/        # Custom React hooks
├── lib/          # Core utilities and helpers
├── pages/        # Page components
├── services/     # Service layer implementations
├── store/        # State management
└── utils/        # Utility functions
```
